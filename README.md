# Tiny-OpenGL-Shadow-Mapping-Examples
Compact OpenGL Shadow Mapping Examples in a single compilation unit.

## Dependencies
* glut (or freeglut)
* glew (Windows only)

## How to compile
Compilation instructions are at the top of each .c file.
